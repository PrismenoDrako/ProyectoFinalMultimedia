CURSO DE PROGRAMACIÓN MUTIMEDIA


SUMILLA

Este es un curso teórico-práctico con un mayor énfasis en la práctica. Este curso pretende formar al estudiante de ingeniería informática en el conocimiento y manejo de la programación multimedia que le permita desarrollar aplicaciones Web y otras que incorporen vídeos, sonidos, animación, gráficos y texto. Los temas que se pretenden abordar permitirán al futuro profesional:

  • Conocer los conceptos básicos y estructura de un lenguaje de programación multimedia
  • Programar correctamente aplicaciones Web utilizando una herramienta de desarrollo de fácil uso.


OBJETIVOS GENERALES


  • Proporcionar los conocimientos de un lenguaje de programación multimedia para desarrollar diversos componentes de un sistema informático multimedia, incluyendo animaciones, sonidos, vídeos, textos y gráficos.
  • Presentar los componentes de un ambiente Web.
  • Desarrollar las habilidades del alumno que le permitan cubrir las necesidades para desarrollar un sistema con una herramienta multimedia.
